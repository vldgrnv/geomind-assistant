# geomind-assistant
# GeoMind — AI-ассистент для ГИС-специалистов

> Цель проекта: создание интеллектуального веб-сервиса, который помогает ГИС-специалистам быстро находить и применять проверенные алгоритмы обработки пространственных данных через естественно-языковые запросы.

Проект разрабатывается в рамках дипломной работы по специальности «Картография и геоинформатика» и поддержан грантом на стартап-разработку.

---

## Принцип работы:
1. Пользователь (ГИС-специалист) пишет запрос в чат. 
2. Система ищет в базе подходящий алгоритм  
3. Программа формирует промпт для нейросети, прикрепляя найденный алгоритм.  
4. Нейросеть генерирует пошаговый ответ и пользователь получает готовую инструкцию.
